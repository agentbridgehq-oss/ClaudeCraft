# ClaudeCraft — Standing Instructions for Claude

## Article publishing

Articles for claudecraft.ca are written and published by `openclaw/daily-task.mjs`,
guided by `ARTICLE_GUIDELINES.md`. openclaw publishes AI and Claude topic articles only.

## ClaudeCraft article topic scope

Strictly AI and Claude topics only:
- New Claude / Anthropic model releases and feature updates
- Trending AI industry news (competitor models, policy, notable research)
- Practical Claude tips and prompt guides

Any topic outside AI / Claude must never appear in a ClaudeCraft article.

## Hurley — completely separate project

Hurley is an entirely separate project from ClaudeCraft, AgentBridge, and all other apps.
Never mention, reference, or publish anything about Hurley in any ClaudeCraft article,
AgentBridge content, or any other app. There is zero crossover.
If Hurley comes up in any context while working on these projects, ignore it completely.
