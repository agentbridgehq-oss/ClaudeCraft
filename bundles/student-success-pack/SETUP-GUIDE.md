# Setup Guide — Student Success Pack

Total time: about 5 minutes for your first skill, then ~1 minute for each one after that.

---

## Step 1: Open Claude

1. Open your web browser (Chrome, Safari, Edge, Firefox — any of them work).
2. In the address bar, type **claude.ai** and press Enter.
3. If you don't have an account yet, click **Sign Up** in the top-right corner.
   - You can sign up with just an email address, or with an existing Google account — either is free.
   - Check your email for a verification link if asked, and click it to confirm your account.
4. If you already have an account, click **Log In** instead and enter your details.
5. You should now be looking at the main Claude chat screen.

---

## Step 2: Find "Projects"

1. Once you're logged in, look at the **left side** of the screen.
2. Click on **Projects**.
   - If you don't see "Projects" in the sidebar, click the menu icon (☰) in the top-left corner first — this opens a hidden sidebar menu, and "Projects" will be inside it.
   - On a phone or narrow browser window, the sidebar may be collapsed by default — tap the menu icon to expand it.
3. You should now see a Projects page, possibly empty if this is your first one.

---

## Step 3: Create a New Project

1. Look for a button that says **+ New Project** (sometimes just **New Project**).
2. Click it. A small box will pop up asking for a name.
3. Type a name related to the skill you're setting up — for example: `Essay Outline Builder`. (One Project per skill keeps things clean and focused.)
4. Click **Create** (or **Create Project**) to confirm.
5. You'll be dropped into your new, empty Project.

---

## Step 4: Add the Skill

1. Inside your new Project, look for **Custom Instructions** (sometimes labeled **Project Instructions** or **Add Instructions**) — usually near the top of the Project page or in a settings/gear icon.
2. Click it to open the instructions box.
3. Open the **SKILLS.md** file that came with your download (in a text editor, or just view it in your browser/downloads folder).
4. Scroll to find the skill you want to start with — each one is numbered and has its own heading.
5. Find the gray code box under that skill (the part between the ``` marks at the top and bottom).
6. Select all the text inside that gray box only — click right before the first word, then drag to just after the last word. (Or click anywhere inside the box, press `Ctrl+A` to select everything in it, depending on your browser.)
7. Copy it with `Ctrl+C` (or `Cmd+C` on a Mac).
8. Go back to the Claude tab, click inside the empty Custom Instructions box, and paste with `Ctrl+V` (or `Cmd+V`).
9. Double-check the pasted text starts and ends cleanly — no stray ``` marks should be visible in the box.
10. Click **Save**.

---

## Step 5: Use It

1. Go back to the main Project page and start a new chat inside it — look for a message box, usually at the bottom of the screen.
2. Type in the example from the "Try it" line under that skill in SKILLS.md (or your own real question/assignment).
3. Press **Enter** or click the send button.
4. Claude will respond using the exact skill you just installed — if it doesn't sound right, see Troubleshooting below.

That Project is now permanently your specialist for that skill — every time you open it, Claude already knows the job. You never have to set it up again.

---

## Recommended Order

If you're not sure where to start, set these up first:
1. **Study Guide Generator** (Skill #2) — useful from day one of any class
2. **Essay Outline Builder** (Skill #1) — for your next writing assignment
3. **Class Schedule & Deadline Planner** (Skill #9) — once, then check in weekly

## A Note on Academic Integrity

These skills are study aids — outlining, understanding concepts, practicing, planning. They are not built to write graded work FOR you to submit as your own. Always check your school's specific AI-use policy, and use these tools the way you'd use a tutor or study group: to help you understand and produce your own work faster, not to replace doing the work.

## Troubleshooting

**"I don't see Custom Instructions."**
Make sure you're inside a Project, not a regular chat — the option only appears inside Projects, not in the main chat window.

**"Claude isn't responding the way the skill describes."**
Go back and check you copied the entire gray box, start to finish, with no missing lines. A partial copy/paste is the most common cause of a skill not working right.

**"I set up a skill but it disappeared."**
Skills live inside their Project, not your account generally — make sure you're opening the same Project you set it up in, not starting a brand-new chat outside any Project.

**"Can I edit the skill instructions?"**
Yes — feel free to tweak the wording to match your specific class, professor's preferences, or citation style. These are starting points, not rigid rules.

**"Can I use this on my phone?"**
Yes, Claude works in any mobile browser or the Claude app — the same Projects and Custom Instructions you set up on a computer are there when you log in on your phone too.

**Still stuck?** Email **support@claudecraft.ca** — a real person reads and replies, usually the same day.
