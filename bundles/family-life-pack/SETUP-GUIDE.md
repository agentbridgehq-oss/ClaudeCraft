# Setup Guide — Family Life Pack

Total time: about 5 minutes for your first skill, then ~1 minute for each one after that.
You do not need to install any software. Everything happens inside your web browser.

---

## Step 1: Open Claude

1. Go to **claude.ai** in your web browser.
2. If you don't have an account yet, click **Sign Up** and create one — it's free.
3. If you already have an account, click **Log In**.

---

## Step 2: Find "Projects"

1. Once you're logged in, look at the **left side** of the screen.
2. Click on **Projects**.
   - If you don't see "Projects" in the sidebar, click the menu icon (☰) in the top-left first to open the sidebar.

---

## Step 3: Create a New Project

1. Click the button that says **+ New Project** (or just **New Project**).
2. Name it after the skill — for example: `Family Meal Planner`.
3. Click **Create** (or **Create Project**).

---

## Step 4: Add the Skill

1. Inside your new project, find **Custom Instructions** (sometimes labeled **Project Instructions**).
2. Click it.
3. Open the **SKILLS.md** file that came with your download.
4. Find **Skill #1: Family Meal Planner** (or whichever skill you're starting with).
5. Copy the text inside the gray box — the part between the ``` marks only.
6. Paste that text into the Custom Instructions box in Claude.
7. Click **Save**.

---

## Step 5: Use It

1. Start a new chat inside that project.
2. Type your situation in plain language — use the "Try it" example under each skill if you want a fast first test.
3. Claude will respond using the skill you just installed.

That project is now permanently your Family Meal Planner specialist (or whichever skill you set up). Every time you open it, Claude already knows the job.

---

## Setting Up the Rest of Your 10 Skills

Repeat Steps 3-5 for each skill — **one skill per project**. Each works best with its own dedicated space rather than mixed together.

Most parents start with whichever skill matches this week's actual chaos:
- Dinner is the daily struggle → **Family Meal Planner**
- A hard conversation is coming up → **Tough Conversation Script Writer**
- Schedules are colliding → **Family Calendar & Logistics Coordinator**
- Homework time is a nightly fight → **Age-Appropriate Homework Helper**

There's no wrong order — start with whatever's loudest right now.

---

## A Note on Privacy

You never need to share your child's full name, school, address, or other identifying details for any of these skills to work well. "My 8-year-old" and a brief description is always enough. Treat it the way you'd treat a conversation with a knowledgeable friend, not a school record.

---

## Troubleshooting

**"I don't see a Custom Instructions option."**
Make sure you're inside a Project, not a regular chat. The option only appears inside Projects.

**"Claude isn't responding the way the skill describes."**
Double-check you copied the entire block of text, start to finish — a partial copy/paste is the most common cause.

**"The advice doesn't fit my exact family situation."**
Tell Claude what's different, in your own words. These skills are starting points built to adapt — correcting it makes the next answer better.

**"Can I edit the skill instructions?"**
Yes. Tweak the wording to fit your family's actual values and rules — these are starting points, not rigid scripts.

**Still stuck?** Email **support@claudecraft.ca** — we'll get you sorted, usually same day.
