# Setup Guide — Job Seeker's Career Pack

Total time: about 5 minutes for your first skill, then ~1 minute for each one after that.

---

## Step 1: Open Claude

1. Open your web browser and go to **claude.ai**.
2. If you don't have an account yet, click **Sign Up** — you can use an email address or an existing Google account, and it's free.
3. Check your email for a verification link if asked, and click it to confirm.
4. If you already have an account, click **Log In** instead.
5. You're now looking at the main Claude chat screen.

---

## Step 2: Find "Projects"

1. Look at the **left side** of the screen.
2. Click **Projects**.
   - If you don't see it, click the menu icon (☰) in the top-left first to open the sidebar — "Projects" will be inside it.

---

## Step 3: Create a New Project

1. Click **+ New Project** (or **New Project**).
2. A box will pop up asking for a name — type something tied to the skill, for example: `Resume Tailoring`.
3. Click **Create** to confirm.
4. You're dropped into your new, empty Project.

---

## Step 4: Add the Skill

1. Inside the Project, find **Custom Instructions** (sometimes labeled **Project Instructions**).
2. Click it to open the instructions box.
3. Open the **SKILLS.md** file from your download.
4. Find the skill you want to start with — each is numbered with its own heading.
5. Select the text inside the gray code box (between the ``` marks) — click before the first word, drag to after the last word, or click inside and press `Ctrl+A`.
6. Copy it with `Ctrl+C` (`Cmd+C` on Mac).
7. Switch back to Claude, click inside the empty Custom Instructions box, and paste with `Ctrl+V` (`Cmd+V`).
8. Confirm nothing got cut off — the text should read cleanly start to finish.
9. Click **Save**.

---

## Step 5: Use It

1. Start a new chat inside that same Project.
2. Try the example under the skill in SKILLS.md, or paste in a real job posting/resume of your own.
3. Press Enter or click send.
4. Claude responds using exactly that skill — that Project is now permanently your specialist for it.

---

## Recommended Order

If you're actively job searching right now, set these up first:
1. **Resume Tailoring for a Specific Job** (Skill #2) — use it on your very next application
2. **Cover Letter Writer** (Skill #3)
3. **Interview Question Prep Coach** (Skill #4) — once you land an interview
4. **Job Search Tracker & Follow-Up Reminder** (Skill #7) — set up once, use weekly

## A Note on Honesty

Every skill in this bundle is built to help you present your REAL experience more effectively — not to fabricate experience or credentials you don't have. Hiring managers and background checks catch this, and it's never worth the risk. Use these tools to communicate your actual strengths clearly and confidently.

---

## Troubleshooting

**"I don't see Custom Instructions."**
Make sure you're inside a Project, not a regular chat — it only appears inside Projects.

**"Claude isn't responding the way the skill describes."**
Check you copied the whole gray box, start to finish. A partial copy/paste is the most common fix.

**"Can I tweak the wording?"**
Yes — adjust tone, formatting, or specifics to match your industry. These are starting points, not rigid scripts.

**"Should I use one Project per application, or one for everything?"**
One Project per SKILL (Resume Tailoring, Cover Letter, etc.), reused across every application — not one per job posting. Paste the new job details into the same Project each time.

**"Can I use this on my phone?"**
Yes — Claude works in any mobile browser or app, and your Projects/skills are there when you log in.

**Still stuck?** Email **support@claudecraft.ca** — a real person replies, usually the same day.
