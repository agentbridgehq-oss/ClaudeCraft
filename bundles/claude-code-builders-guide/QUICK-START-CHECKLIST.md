# Quick-Start Checklist — Claude Code Builder's Guide

Print this or keep it open in a second tab while you go through the main guide.

## First 30 Minutes
- [ ] Chose Pro/Max subscription OR API billing (Part 2)
- [ ] Installed VS Code
- [ ] Installed Node.js (LTS version)
- [ ] Ran `npm install -g @anthropic-ai/claude-code` in VS Code's terminal
- [ ] Logged in via `claude` and approved it in the browser

## First Session
- [ ] Built the Hello World page (Part 4)
- [ ] Opened the file it created and actually read it
- [ ] Asked for one change in plain English and saw it applied
- [ ] Viewed the result in a real browser

## First Real Project
- [ ] Described one real feature, one at a time, not five at once
- [ ] Reviewed every change before trusting it (Part 5.1.c)
- [ ] Created a free GitHub account and pushed code to a repo
- [ ] Created a free Railway account and asked Claude Code to deploy it

## If You Get Stuck
1. Re-read the matching Part — it's organized so you can jump straight to the relevant Step.
2. Describe exactly what's wrong, in plain English, to Claude Code directly — "that's not working, here's what I see" is a completely valid request.
3. Email support@claudecraft.ca — include what you tried and what happened. A real person replies.
