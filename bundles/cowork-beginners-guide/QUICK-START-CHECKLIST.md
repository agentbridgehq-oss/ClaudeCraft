# Quick-Start Checklist — Claude Co-Work Beginner's Guide

Print this or keep it open in a second tab while you go through the main guide.

## First 30 Minutes
- [ ] Created my free claude.ai account
- [ ] Sent my first message in a New Chat
- [ ] Created my first Project (named after a real job, not "Project 1")
- [ ] Added Custom Instructions to that Project
- [ ] Started a chat from inside the Project to confirm the instructions are active

## First Week
- [ ] Used the Role / Task / Context / Format prompt structure at least 3 times
- [ ] Iterated on at least one response instead of starting a new chat from scratch
- [ ] Uploaded one real file (document, spreadsheet, or PDF) and asked Claude to do something useful with it
- [ ] Pasted a sample of my own writing into a Project and asked Claude to match the style

## First Month
- [ ] Have at least 3 active Projects covering my most repeated tasks
- [ ] Ran one full "Monday plan → Friday review" cycle (Part 7 of the main guide)
- [ ] Can name my own top 2 mistakes from Part 8 and the fix for each

## If You Get Stuck
1. Re-read the matching Part in the main guide — it's organized so you can jump straight to the relevant Step.
2. Try rephrasing using Role / Task / Context / Format.
3. Email support@claudecraft.ca — include what you tried and what happened. A real person replies.
