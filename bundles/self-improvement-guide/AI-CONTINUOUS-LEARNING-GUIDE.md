# AI Agent Continuous Learning Guide — A System, Not Just Prompts

A repeatable weekly loop for using Claude as a standing self-improvement partner — not a one-off "give me advice" chat you never return to.

Email **support@claudecraft.ca** if you get stuck.

---

## Setup (run once, save as Custom Instructions in a dedicated Project)

```
Act as my ongoing self-improvement accountability partner. Interview me one question at a time: (1) my top 3 goals right now, (2) what's worked before, (3) what usually makes me fall off a habit, (4) how direct vs. gentle you should be. After all 4, summarize it into a profile I'll save as your permanent instructions here.
```

## The Weekly Review (the actual habit — same prompt, same Project, every week)

```
Weekly review. This week: [2-3 sentences — wins, misses, anything notable].
1. Compare this to my stated goals.
2. Be honest if I'm drifting — don't be falsely encouraging.
3. Give me ONE specific focus for next week, not five.
```

## Daily Check-In (optional, for building a new habit fast)

```
Quick check: did I do [habit] today? [yes/no + one line why]. If I broke a streak, ask one diagnostic question about what got in the way — no lecture.
```

## Monthly Skill-Gap Finder

```
Based on my goals and recent weeks, what's the ONE gap that, if fixed, would have the biggest knock-on effect on everything else? Be specific, not generic.
```

## Reframe Prompt (for motivation dips)

```
I'm losing motivation on [goal]. Before telling me to push through, ask what specifically changed recently that made it harder than when I started.
```

This works because each prompt does one job inside a Project that already remembers your real history — the advice gets more specific over time, not less.
