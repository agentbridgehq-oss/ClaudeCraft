# Setup Guide — Claude Power User Pack

Total time: about 5 minutes for your first skill, then ~1 minute for each one after that.
You do not need to install any software. Everything happens inside your web browser.

---

## Step 1: Open Claude

1. Go to **claude.ai** in your web browser.
2. If you don't have an account yet, click **Sign Up** and create one — it's free.
3. If you already have an account, click **Log In**.

---

## Step 2: Find "Projects"

1. Once you're logged in, look at the **left side** of the screen.
2. Click on **Projects**.
   - If you don't see "Projects" in the sidebar, click the menu icon (☰) in the top-left first to open the sidebar.

---

## Step 3: Create a New Project

1. Click **+ New Project** (or just **New Project**).
2. Name it after the skill — for example: `Build a Claude Skill`.
3. Click **Create**.

---

## Step 4: Add the Skill

1. Inside your Project, find **Custom Instructions**.
2. Click it.
3. Open the **SKILLS.md** file that came with your download.
4. Find the skill you want to start with.
5. Copy the text inside the gray box — the part between the ``` marks only.
6. Paste it into the Custom Instructions box.
7. Click **Save**.

---

## Step 5: Use It

1. Start a new chat inside that project.
2. Use the "Try it" example under each skill for a fast first test.
3. Claude will respond using the skill you just installed.

---

## A Note on Skill #1 (Build a Real Claude Skill)

This one is different from the other 7 — it walks you through packaging a Custom Instructions prompt into an actual native Claude Skill (the zip + SKILL.md format). Native Skills now work on **every plan, including Free** — the only real requirement is turning on **Code Execution**:

1. Go to **Settings → Capabilities**.
2. Turn on **"Code execution and file creation."**
3. Once that's on, go to **Customize → Skills** to upload your packaged zip.

Skills #2-8 don't need Code Execution at all — they work exactly like every other ClaudeCraft skill, paste-and-go.

---

## Setting Up the Rest of Your 8 Skills

Repeat Steps 3-5 for each skill — one skill per Project. Most power users start with whichever skill solves what's actively annoying them: **Token-Saving Prompt Auditor** if you're hitting usage limits, **Code Review Specialist** if you're shipping code, **Multi-Step Agentic Workflow Builder** if you're automating something repetitive.

---

## Troubleshooting

**"I don't see a Custom Instructions option."**
Make sure you're inside a Project, not a regular chat. The option only appears inside Projects.

**"I don't see Code Execution in Settings."**
Code Execution availability can vary slightly by plan and region rollout — if it's genuinely missing, every other skill in this pack still works without it.

**"Claude isn't responding the way the skill describes."**
Double-check you copied the entire block of text, start to finish.

**Still stuck?** Email **support@claudecraft.ca** — we'll get you sorted, usually same day.

---

## A Note on Claude Plans

7 of these 8 skills work on the **free** Claude plan. Only Skill #1 (Build a Real Claude Skill) needs Code Execution turned on — available on Free, Pro, Max, Team, and Enterprise plans.
