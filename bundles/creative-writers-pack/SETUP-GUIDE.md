# Setup Guide — Creative Writer's Pack

Total time: about 5 minutes for your first skill, then ~1 minute for each one after that.
You do not need to install any software. Everything happens inside your web browser.

---

## Step 1: Open Claude

1. Go to **claude.ai** in your web browser.
2. If you don't have an account yet, click **Sign Up** and create one — it's free.
3. If you already have an account, click **Log In**.

---

## Step 2: Find "Projects"

1. Once you're logged in, look at the **left side** of the screen.
2. Click on **Projects**.
   - If you don't see "Projects" in the sidebar, click the menu icon (☰) in the top-left first to open the sidebar.

---

## Step 3: Create a New Project

1. Click **+ New Project** (or just **New Project**).
2. Name it after the skill — for example: `Story Outline Builder`.
3. Click **Create**.

We recommend creating one Project **per manuscript**, not per skill, once you're using several skills on the same story — see "A Note on Organizing by Project" below.

---

## Step 4: Add the Skill

1. Inside your Project, find **Custom Instructions**.
2. Click it.
3. Open the **SKILLS.md** file that came with your download.
4. Find the skill you want to start with.
5. Copy the text inside the gray box — the part between the ``` marks only.
6. Paste it into the Custom Instructions box.
7. Click **Save**.

---

## Step 5: Use It

1. Start a new chat inside that project.
2. Paste in your story material, or use the "Try it" example under each skill for a fast first test.
3. Claude will respond using the skill you just installed.

---

## A Note on Organizing by Project

Unlike our other bundles (where one skill = one Project usually makes sense), creative writing often works better with **one Project per manuscript**, containing the full context of that story. If you want multiple skills (Outline Builder, Consistency Tracker, Beta Reader) all aware of the same story, you can combine their Custom Instructions into a single Project for that book — just separate each skill's instructions clearly with a heading when pasting them in together.

If you write across multiple unrelated projects/books at once, separate Projects per manuscript keeps each story's world and characters from bleeding into another.

---

## Setting Up the Rest of Your 10 Skills

Repeat Steps 3-5 for each skill you want active. Most writers don't use all 10 at once — pick based on where you are right now:

- Starting a new story → **Story Outline Builder**, **Character Developer**
- Deep in a draft → **Writer's Block Buster**, **World-Building Consistency Tracker**
- Revising → **Dialogue Polisher**, **Line-Edit Pass Assistant**, **Beta-Reader Feedback Simulator**
- Finished and submitting → **Query Letter & Pitch Writer**, **Synopsis & Back-Cover Copy Writer**

---

## Troubleshooting

**"I don't see a Custom Instructions option."**
Make sure you're inside a Project, not a regular chat. The option only appears inside Projects.

**"Claude isn't responding the way the skill describes."**
Double-check you copied the entire block of text, start to finish.

**"I want to combine two skills in one Project — how?"**
Paste both skill blocks into the same Custom Instructions field, each under its own short heading (like "## Outline Builder" and "## Consistency Tracker"), so Claude can tell which job you're asking for in a given message.

**Still stuck?** Email **support@claudecraft.ca** — we'll get you sorted, usually same day.
