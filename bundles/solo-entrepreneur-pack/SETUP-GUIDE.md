# Setup Guide — Solo Entrepreneur Pack

Total time: about 5 minutes for your first skill, then ~1 minute for each one after that.
You do not need to install any software. Everything happens inside your web browser.

---

## Step 1: Open Claude

1. Open your web browser and go to **claude.ai**.
2. If you don't have an account yet, click **Sign Up** and create one — email or an existing Google account both work, and it's free.
3. Check your email for a verification link if asked, and click it to confirm.
4. If you already have an account, click **Log In** instead.
5. You're now on the main Claude chat screen.

---

## Step 2: Find "Projects"

1. Once you're logged in, look at the **left side** of the screen.
2. Click on **Projects**.
   - If you don't see "Projects" in the sidebar, click the menu icon (☰) in the top-left first to open the sidebar.
   - On mobile or a narrow window, the sidebar may start collapsed — tap the menu icon to expand it.

---

## Step 3: Create a New Project

1. Click the button that says **+ New Project** (or just **New Project**).
2. Type a name for this project. We recommend naming it after the skill, for example: `Proposal Writer`.
3. Click **Create** (or **Create Project**).
4. You're dropped into your new, empty Project.

---

## Step 4: Add the Skill

1. Inside your new project, look for a button or section called **Custom Instructions** (sometimes labeled **Project Instructions** or **Add Instructions**).
2. Click it.
3. Open the file **SKILLS.md** that came with your download.
4. Find **Skill #1: Proposal Writer** (or whichever skill you want to start with) — each skill has its own numbered heading.
5. Find the gray code box under that skill (the part between the ``` marks).
6. Select all the text inside that box only — click before the first word and drag to after the last word, or click inside and press `Ctrl+A`. Don't copy the headings around it.
7. Copy it with `Ctrl+C` (`Cmd+C` on Mac).
8. Switch back to Claude, click inside the empty Custom Instructions box, and paste with `Ctrl+V` (`Cmd+V`).
9. Check the pasted text reads cleanly, start to finish, with no stray ``` marks.
10. Click **Save**.

---

## Step 5: Use It

1. Start a new chat inside that project — look for the message box near the bottom of the screen.
2. Type your request — use the "Try it" example under each skill in SKILLS.md if you want a fast first test.
3. Press Enter or click send.
4. Claude will respond using the skill you just installed.

That's it — that project is now permanently your Proposal Writer specialist. Every time you open it, Claude already knows what to do. You never have to set it up again.

---

## Setting Up the Rest of Your 15 Skills (+1 Bonus)

Repeat Steps 3-5 for each skill you want — **one skill per project**. Don't combine multiple skills into the same project; each works best with its own dedicated space.

We recommend starting with whichever skill solves your most immediate problem right now. Most customers start with **Proposal Writer**, **Client Email Machine**, or **Invoice Follow-Up System** — pick whichever matches what's on your plate today. Save the bonus skill for once you're comfortable with the core 15.

---

## Troubleshooting

**"I don't see a Custom Instructions option."**
Make sure you're inside a Project (not just a regular chat). The option only appears inside Projects.

**"Claude isn't responding the way the skill describes."**
Double-check you copied the entire block of text, including the first and last lines. A partial copy/paste is the most common cause.

**"Can I edit the skill instructions?"**
Yes — feel free to tweak wording to better match your business. These are starting points, not rigid rules.

**"Should I make a new Project for every client, or reuse one?"**
Reuse one Project per skill (e.g. one "Client Email Machine" Project), not one per client — just give Claude the specific client's context each time you chat.

**"Can I use this on my phone?"**
Yes — Claude works in any mobile browser or app, with the same Projects and skills available wherever you log in.

**Still stuck?** Email **support@claudecraft.ca** — we'll get you sorted, usually same day.
